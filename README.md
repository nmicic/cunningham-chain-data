# Cunningham Chain Dataset

First-kind Cunningham chain roots (CC10 through CC17) from the CC18 search campaign.

## What this repo contains

| Path | Description |
|------|-------------|
| `sample/cc10plus_roots_sample_1000.txt` | First 1000 roots (raw text format) |
| `sample/cc10plus_roots_sample_1000.csv` | Same 1000 roots (normalized CSV) |
| `scripts/make_snapshot_csv.py` | Converts raw text to normalized CSV |
| `manifest/dataset_manifest.json` | Snapshot metadata and counts |
| `schema/README.md` | Format documentation for both raw and CSV |

## Full dataset

The full dataset is available as GitHub Release assets (not tracked in git):

| Asset | Download | Uncompressed | Format |
|-------|----------|--------------|--------|
| `cc10plus_roots_snapshot_2026-03-12.txt.gz` | 11 MB | 30 MB | `CC<level> 0x<hex> <digits>` |
| `cc10plus_roots_snapshot_2026-03-12.csv.gz` | 12 MB | 32 MB | `cc,root_hex,digits,bits` |

Pre-built assets are in `.release_assets/` for upload during the first GitHub Release.

The raw text file is the canonical dataset. The CSV is a normalized convenience copy generated by `scripts/make_snapshot_csv.py`.
